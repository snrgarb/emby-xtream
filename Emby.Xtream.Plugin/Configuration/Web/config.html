Dispatcharr Integration

Dispatcharr is an open-source IPTV stream proxy. When enabled, it applies to Live TV, VOD movies, and series: faster channel switching via FFprobe bypass, stream proxying, reconnect handling, and multiple quality versions per movie (when multiple provider streams exist).

Streamflow advantage: Per-channel codec and resolution metadata is only available when Streamflow is enabled and configured inside Dispatcharr. Without it, stream stats will not be populated and FFprobe bypass will not apply.
Route Live TV, VOD movies, and series through Dispatcharr for FFprobe bypass, stream proxying, and multi-version STRM support.
These are your Dispatcharr web UI login credentials — the account you use to sign in to the Dispatcharr admin interface. They are not the same as the Xtream API username/password you may have set in the Xtream Connection section above.
